Навигационный котроллер

Для дистанционного наблюдения за подвижными объектами, может быть использован совместно с любым совместимым программным комплексом.

Характеристики Ресурсы

Особенности Arnavi 6:

  • Модем GSM с поддержкой сетей 2G и 4G;
  • Вариативность по антеннам GSM и навигации: встроенная или внешняя;
  • Встроенный WIFI модуль – передача данных на сервер (основной или резервный канал). Опциональное оснащение в виде платы расширения Данный функционал можно прописывать в тендерах для ограничения других участников;
  • Поддержка Bluetooth 4.0 (BLE) – подключение беспроводных датчиков (ДУТ, термодатчики и другие);
  • Большой набор цифровых интерфейсов: CAN, RS485, RS232, 1-WIRE;
  • Подключение по CAN ко многим ТС напрямую (список машин постоянно обновляется);
  • Качественная защита по питанию до 500 В;
  • Специализированная мс для заряда и контроля АКБ. Можно настраивать режимы зарядки, контроль заряда от температуры;
  • Сервисная кнопка на плате для функции скачать настройки, обновить ПО с WEB конфигуратора. Это удобно, когда нет под рукой ПК, то простым нажатием на кнопку прибор сам загружает назначенную версию ПО и настройки;
  • Настройка, диагностика через мобильное приложение (варианты подключения Bluetooth, USB);
  • Подключение проводной двухсторонней громкой связи;
  • Большое кол-во дискретных входов и выходов под различные задачи Данный функционал можно прописывать в тендерах для ограничения других участников;
  • Выходы питания для датчиков 5 В и 12 В Данный функционал можно прописывать в тендерах для ограничения других участников.
